# E-Ticaret Uygulaması

## Proje Amacı
Bu proje, kullanıcıların çevrimiçi olarak ürünleri inceleyip satın alabileceği, yöneticilerin ise ürün ve kategorileri kolayca yönetebileceği modern ve ölçeklenebilir bir e-ticaret platformu sunmayı amaçlamaktadır.

## Hedef Kullanıcı Kitlesi
*   **Son Kullanıcılar (Müşteriler):** Online alışveriş yapmak, ürünleri incelemek ve sipariş vermek isteyen internet kullanıcıları.
*   **Site Yöneticileri (Admin):** E-ticaret platformundaki ürün, kategori ve sipariş süreçlerini yöneten yetkililer.

## Senaryo / Kullanım Amacı
Bu proje, tipik bir e-ticaret senaryosunu simüle eder:
1.  **Ürün Keşfi:** Kullanıcılar ana sayfada veya kategoriler altında listelenen ürünleri görüntüler.
2.  **Detay İnceleme:** İlgilendikleri ürünün detay sayfasına giderek resim, fiyat ve açıklama bilgilerini inceler. Giyim ürünleri için uygun beden seçimini yapar.
3.  **Sepet İşlemleri:** Seçtikleri ürünleri sepete ekler, sepet içeriğini günceller veya boşaltır.
4.  **Sipariş ve Ödeme:** Sepetteki ürünleri onaylayarak ödeme adımına geçer ve sipariş işlemini tamamlar.
5.  **Yönetim (Admin):** Yetkili kullanıcılar, admin paneli üzerinden yeni ürünler ekleyebilir, mevcut ürünlerin bilgilerini (stok, fiyat, görsel vb.) güncelleyebilir ve kategori yapısını düzenleyebilir.

## Kullanılan Teknolojiler
Bu projenin geliştirilmesinde aşağıdaki teknolojiler kullanılmıştır:

*   **C#** (Programlama Dili)
*   **ASP.NET Core MVC** (Web Uygulama Çatısı)
*   **SQL Server Express** (Veritabanı Yönetim Sistemi)
*   **Entity Framework Core** (ORM - Veri Erişimi)
*   **HTML / CSS / Bootstrap** (Ön Yüz Tasarımı)
